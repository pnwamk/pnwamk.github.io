I'm a researcher and instructor at Indiana University advised by Sam Tobin-Hochstadt.

My research interests revolve around programming languages and software engineering; in particular I frequently look for ways to leverage types, type theory, and formal verification techniques to build more robust systems.

Some of my work has involved an extension to Typed Racket which adds support for basic dependent types. You can try it out by downloading Racket here (version >= 6.11).

Select Project Involvement

  • Racket - I frequently program in and ocassionally contribute to the programmable programming language Racket.
  • Typed Racket - I help maintain and improve Typed Racket (a typed dialect of Racket) and have added several features to the language (e.g. refinement types).
  • Pie - I have made a few contributions to the dependently typed language Pie and have a small language extension (the Racket pie-a-let-mode package) which adds a few features which are useful for learning/teaching the basics of dependent types.
